# GALLERY.3D.COLUMN

Changes the format of the active chart to a 3-D column chart.

**Syntax**

**GALLERY.3D.COLUMN**(**type\_num**)

**GALLERY.3D.COLUMN**?(type\_num)

Type\_num&nbsp;&nbsp;&nbsp;&nbsp;is the number of the 3-D Column format
that you want to apply to the chart.
