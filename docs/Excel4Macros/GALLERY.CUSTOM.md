GALLERY.CUSTOM

Changes the format of the active chart to the custom format.

**Syntax**

**GALLERY.CUSTOM**(**name\_text**)

Name\_text**&nbsp;&nbsp;&nbsp;nbsp;&nbsp;&nbsp;&nbsp;nbsp;&nbsp;&nbsp;&nbsp;nbsp;&nbsp;is the name of the custom template you
want to apply.
