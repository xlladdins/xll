SHOW.ACTIVE.CELL

Scrolls the active window so the active cell becomes visible. If an
object is selected, SHOW.ACTIVE.CELL returns the \#VALUE\! error value
and halts the macro.

**Syntax**

**SHOW.ACTIVE.CELL**( )

**Related Functions**

[ACTIVE.CELL](ACTIVE.CELL.md)   Returns the reference of the active cell
