# GALLERY.3D.LINE

Changes the format of the active chart to a 3-D line chart.

**Syntax**

**GALLERY.3D.LINE**(**type\_num**)

**GALLERY.3D.LINE**?(type\_num)

Type\_num&nbsp;&nbsp;&nbsp;&nbsp;is the number of the 3-D Line format
that you want to apply to the chart.
